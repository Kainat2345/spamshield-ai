# 📧 SpamShield AI — SaaS Email Protection System

## 🚀 Overview

SpamShield AI is a **machine learning-based SaaS-style web app** that detects whether an email/message is **Spam or Not Spam** using Natural Language Processing (NLP).

It is built with **Python, Scikit-learn, and Streamlit** and designed with a **modern SaaS dashboard UI**.

---

## ✨ Features

* 📧 Spam vs Not Spam prediction
* 🧠 AI-powered NLP model (TF-IDF + Naive Bayes)
* 📊 Confidence score for predictions
* 📜 Prediction history tracking
* 📥 Downloadable report (CSV)
* 🎨 SaaS-style Streamlit UI
* 👤 User session tracking

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas 📊
* Scikit-learn 🤖
* Streamlit 🌐
